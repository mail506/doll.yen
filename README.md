# 💱 USD/JPY 為替分析ダッシュボード

ドル円（USD/JPY）に特化したリアルタイム為替分析ダッシュボードです。
yfinance 経由でリアルタイムデータを取得し、テクニカル・ファンダメンタルズ分析を統合表示します。

---

## セットアップ手順

### 1. 依存ライブラリのインストール

```bash
pip install -r requirements.txt
```

### 2. アプリの起動

```bash
streamlit run app.py
```

ブラウザで `http://localhost:8501` が自動で開きます。

---

## 機能一覧

### ① 総合センチメント・メーター
- RSI / MACD / SMA乖離 / フェアバリュー乖離 / 金利差を総合スコアリング
- 0〜100のスコアで「強気〜弱気」を5段階表示

### ② メトリクスダッシュボード
- 現在値・前日比・52週高安値
- RSI (14)・HV 20日・米10年債・日米金利差

### ③ インタラクティブチャート
- **タブ1**: ローソク足 + SMA20/200 + フェアバリューライン + MACD + RSI
- **タブ2**: ドル円 vs 日米金利差の相関チャート（デュアルY軸） + 相関係数バーチャート

### ④ 予測シナリオカード（1ヶ月）
- **メインシナリオ (65%)**: トレンド継続・±1σレンジ内
- **サブシナリオ (25%)**: 金利差縮小・トレンド転換
- **リスクシナリオ (10%)**: ±2σ超・テールリスク

### ⑤ フェアバリュー分析
- 180日間の重回帰分析（日米金利差・DXY・S&P500を独立変数）
- 現在値との乖離幅・割安/適正/割高の判定

### ⑥ ボラティリティ予測レンジ
- 20日HVベースの1週間・1ヶ月予測レンジ（±1σ/±2σ）

### ⑦ ファンダメンタルズ・シミュレーター
- NFP / CPI / FFRのスライダーで感応度分析
- ウォーターフォールチャートで各指標の影響を可視化

## サイドバー設定

| 設定項目 | 説明 |
|---|---|
| 分析期間 | 3ヶ月 / 6ヶ月 / 1年 |
| 日本10年債利回り | yfinanceで取得困難なため手動入力（推奨） |
| 自動更新 | 5分ごとにデータ再取得 |
| 今すぐ更新 | キャッシュクリア＆即時再取得 |
